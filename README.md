# Square Fill Loader

A small, lightweight square fill loader built with HTML and CSS.

![Preview of the loader](./Screenshot%202024-07-09%20084151.png)

## About

This repository contains a simple CSS loader animation that fills a square. It's easy to drop into any project and customize via CSS.

## Preview

Open `index.html` in your browser to see the loader in action.

## Features

- Pure HTML and CSS (no JavaScript required)
- Small and portable
- Easy to customize color, size, and speed

## Usage

1. Clone the repository or download the files:

```bash
git clone https://github.com/BinaryVortex/Square-Fill-Loader.git
```

2. Open `index.html` in your browser.

3. To embed the loader in another project, copy the HTML from `index.html` and the relevant rules from `style.css`.

Example HTML (from `index.html`):

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link rel="stylesheet" href="style.css">
    <title>Square Fill Loader</title>
  </head>
  <body>
    <div class="loader"></div>
  </body>
</html>
```

Customize colors, size, and animation timing in `style.css` to match your project's look-and-feel.
